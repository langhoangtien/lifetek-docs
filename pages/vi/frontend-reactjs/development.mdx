## Cài Đặt & Phát Triển

### Cài Đặt

1. Tạo một dự án mới:

   ```git copy
   git clone https://g.lifetek.vn/admin/frontend

   ```

2. Cài đặt các thư viện cần thiết:

   ```bash copy
   yarn
   ```

### Cấu Hình

Trước khi chạy ứng dụng, hãy tạo một tệp `.env` trong thư mục gốc bằng cách sao chép tệp `.env.example`:

```bash copy
cp .env.example .env
```

Ví dụ về tệp `.env.example` file

```bash copy
VITE_SERVER_URL=https://api.lifetek.dev/api/v1
```

#### Các Biến Môi Trường

- `VITE_SERVER_URL`: URL của API Backend

### Khởi chạy dự án

```bash copy
 yarn dev
```
