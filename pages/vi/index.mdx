## Tính Năng

- [x] Cơ sở Dữ liệu. Hỗ trợ [TypeORM](https://www.npmjs.com/package/typeorm)
- [x] Dữ liệu Giả (Seeding) ([Typeorm Extension](https://www.npmjs.com/package/typeorm-extension)).
- [x] Dịch vụ Cấu hình ([@nestjs/config](https://www.npmjs.com/package/@nestjs/config)).
- [x] Gửi Email ([@nestjs-modules/mailer](https://www.npmjs.com/package/@nestjs-modules/mailer) & [nodemailer](https://www.npmjs.com/package/nodemailer)).
- [x] Đăng nhập và Đăng ký qua Email.
- [ ] Đăng nhập Mạng Xã hội (Apple, Facebook, Google, Twitter).
- [x] Vai trò Quản trị viên và Người dùng.
- [x] Phân trang: Offset và Cursor (Sao chép từ [typeorm-cursor-pagination](https://github.com/benjamin658/typeorm-cursor-pagination) và thêm nhiều tính năng).
- [x] Đa ngôn ngữ/Dịch thuật (I18N) ([nestjs-i18n](https://www.npmjs.com/package/nestjs-i18n)).
- [x] Tải lên Tệp. Hỗ trợ local và Amazon S3.
- [x] Swagger.
- [x] E2E and units tests.
- [x] Docker.
- [x] CI (GitLab CI/CD).
